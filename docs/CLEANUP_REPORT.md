## Current Critical Cleanup Findings

### High

1. The repo contains both active session-based architecture and older legacy route/helper paths.
2. Supabase schema exists in two places with differences.
3. Several empty legacy directories survived only because of `__pycache__` contents.

### Medium

1. The docs folder contains many historical progress files, which makes the important docs harder to find.
2. The frontend app is concentrated in one large page file.
3. Some env/config settings describe infrastructure that is not used by the current main flow.
